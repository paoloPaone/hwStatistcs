Overview of Attack Simulation

Previous Next

In this project, I developed a unified framework in C# using Windows Forms, designed to modularly manage and execute stochastic simulations for various processes (Bernoulli, Random Walk, Poisson, and Brownian Motion) from previous assignments. The framework follows a centralization approach, consolidating the different simulation functionalities into a single, unified class for easier management and integration.
The system allows users to interact through a set of configurable options and parameters, offering customization of the simulations.
